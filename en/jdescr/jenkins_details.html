Jenkins FreeBSD jail image

Jenkins is a free extendable open source continuous integration server

For installing Jenkins CBSD jail use bsdconfig cbsd and Repository submenu, or just follow this steps:

			% cbsd repo action=get sources=img name=jenkins

or

			% cbsd repo action=get sources=img name=jenkins ver=10.1

If you hosted on another ( non 10.1 ) FreeBSD version.

Execute //wf_imghelper_ssi.html">imghelper that asks you for some questions required for the installation of the jail:

% cbsd imghelper jname=jenkins bootstrap=/usr/jails/jails-system/jenkins/bin/bootstrap.sh

where: /usr/jails/ is workdir for CBSD.

A clean Jenkins instance in the jail will be available for access via http://IP:8180/jenkins/
