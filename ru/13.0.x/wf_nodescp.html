Копирование файлов с/на ноды

Команда nodescp

			% node1:remotefile1 localfile1 [ localfile2 node2@:remotefile2 ]

Команда cbsd nodescp позволяет копировать (при наличие id_rsa ключей) файлы на/с удаленных нод. Синтаксис аргументов аналогичен команде scp(1)

При наличие rootkeyfile для ноды, соединение для пользователя root будет предпочитаться пользователю cbsd, что позволит вам записывать файлы на удаленной ноде не только в каталогах, владелец которых cbsd

Пример:

			% cbsd node mode=list
