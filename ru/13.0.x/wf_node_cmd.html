Команды:
% cbsd jcoldmigrate
% cbsd jbackup
% cbsd jrclone
% cbsd jailmapdb
и др.
Часть команд могут требовать наличие ключа удаленной ноды для выполнения своих функций. Некоторый список подобных команд:

  • cbsd jcoldmigrate - холодная миграция клетки с ноды на ноду
  • cbsd jbackup, jrclone - разовая синхронизация или репликация данных клетки on-the-fly на удаленную ноду
  • cbsd jailmapdb - сгенерировать jmap.txt, содержащий записи вида nodename="jail1 jail2 .." - список и расположение клеток
  • cbsd jwhereis - вернет имя ноды, на котором находится данный jail (работает по файлу jmap.txt)
  • cbsd jlogin - попытаться зайти в клетку, если ее нет локально, но присутствует в jmap.txt
  • cbsd rexe - выполнить команду на удаленной ноде
  • cbsd rnode - залогинится в cbsd shell на удаленной ноде
  • ...
