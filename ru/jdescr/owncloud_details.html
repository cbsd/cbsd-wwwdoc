OwnCloud jail

For installing ownCloud cbsd jail just follow this steps (for FreeBSD 10 and above):

			% cbsd repo action=get sources=img name=owncloud

or

			% cbsd repo action=get sources=img name=owncloud ver=10.0

If you hosted on another ( non 10.0 ) FreeBSD version. After image downloading please set correct name and ip address of the jail and run a install script that asks you for FQDN of WEB server and ownCloud:

			% cbsd jconfig jname=owncloud owncloudinstall

After entering FQDN of instance, Install procedure and starting jail the clean ownCloud installation will be available via http://FQDN/owncloud with

default login: admin

and

default password: admin
