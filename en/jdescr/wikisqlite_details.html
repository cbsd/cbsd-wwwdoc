Mediawiki SQLite3 jail

For installing MediaWiki with SQLite3 backend jail use bsdconfig cbsd and Repository submenu, or just follow this steps:

			% cbsd repo action=get sources=img name=wikisqlite

or

			% cbsd repo action=get sources=img name=wikisqlite ver=10.1

If you hosted on another ( non 10.1 ) FreeBSD version.

Execute //wf_imghelper_ssi.html">imghelper that asks you for some questions required for the installation of the jail:

% cbsd imghelper jname=wikisqlite bootstrap=/usr/jails/jails-system/wikisqlite/bin/bootstrap.sh

where: /usr/jails/ is workdir for CBSD.

After entering FQDN of instance, Install procedure and starting jail the clean MediaWiki installation will be available via http://FQDN/
